## üìå Project Description
The Password Strength Analyzer is a Python-based cybersecurity project that evaluates the strength of user-entered passwords. It checks passwords using multiple security rules and classifies them as Weak, Medium, or Strong. This project helps users understand secure password creation and basic authentication security.


#Objectives
- Analyze password strength using security rules
- Identify weak and common passwords
- Improve awareness of strong password practices
- Apply cybersecurity concepts using Python


#Features
- Password length validation
- Uppercase letter detection
- Lowercase letter detection
- Number detection
- Special character detection
- Strength scoring system
- Improvement suggestions


#Technologies Used
- Programming Language: Python
- Library: re (Regular Expressions)
- Tool: Visual Studio Code


#Working Principle
1. User enters a password in the terminal
2. The program checks password length and character types
3. Each valid rule increases the password score
4. Based on total score, password is classified as:
   - Weak
   - Medium
   - Strong
5. Suggestions are shown for weak passwords
